Project overview
The Problem & Solution

Public charging stations are often free, vandalism-prone, and costly to maintain. E-Piso solves this by gating charging access behind a coin mechanism — a proven, low-tech barrier that requires zero connectivity or accounts.

"Insert a valid coin → phone charges for 2 minutes. Insert more coins → time stacks up. Simple, physical, reliable."

The system is built around an ATmega328 microcontroller that monitors a multi-coin sensor, controls relay switches for up to 4 charging ports, and displays remaining time on a 20×4 I²C LCD display.

01

Coin Detection

Multi-coin sensor validates the coin type and sends a digital pulse to the microcontroller.

02

Time Allocation

Each valid coin adds 2 minutes to the charging credit. Credits stack indefinitely before a port is selected.

03

Port Selection

User presses one of 4 buttons to assign the credit to a specific charging port. Relay activates.

04

Countdown & Auto-cutoff

LCD shows reverse countdown. When time reaches 00:00, relay opens and charging stops automatically.

05

Top-Up Anytime

Insert more coins during an active session — time is added to the remaining balance in real-time.

Arduino Code

Core loop handling coin detection, relay switching, and LCD countdown. Full source on Hackster.io.

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 20, 4);

const int relayPins[4]  = { 2, 3, 4, 5 };
const int buttonPins[4] = { 6, 7, 8, 9 };
const int coinSlotPin   = 11;
const unsigned long chargeTimePerCoin = 120; // 2 min in seconds

void loop() {
  // Coin detection with debounce
  if (digitalRead(coinSlotPin) == LOW) {
    delay(50);
    if (digitalRead(coinSlotPin) == LOW) {
      coinsInserted++;
      tone(buzzerPin, 1000, 100); // beep!
    }
  }
  // Button → start charging on selected port
  for (int i = 0; i < 4; i++) {
    if (digitalRead(buttonPins[i]) == LOW && coinsInserted > 0) {
      startCharging(i);
    }
  }
  // Auto-cutoff when time expires
  for (int i = 0; i < 4; i++) {
    if (relayEndTimes[i] > 0 && millis() >= relayEndTimes[i])
      stopCharging(i);
  }
  updateLCD();
}
