How the Circuit Works

A multi-coin sensor detects valid coins and pulses the ATmega328 on digital pin 11. Each valid coin increments the credit counter and triggers a buzzer beep for user feedback.

// Signal Flow COIN_IN → digitalRead(11) == LOW
→ coinsInserted++ → tone(BUZZER)
→ relayEndTime += 120s per coin

When a port button is pressed, the assigned relay closes, powering the USB socket. The MCU counts down in real-time on the LCD. At 00:00, the relay opens and charging stops automatically.

// Auto Cutoff millis() >= relayEndTime[i]
→ digitalWrite(RELAY[i], LOW)
→ Port OFF · LCD cleared

Arduino Source Code

Core loop — coin detection, relay switching, LCD update. Full source on Hackster.io.

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 20, 4);

const int  relayPins[4]  = {2,3,4,5};
const int  buttonPins[4] = {6,7,8,9};
const unsigned long chargeTimePerCoin = 120; // 2 min/coin

void loop() {
  // Coin detection with debounce
  if (digitalRead(coinSlotPin) == LOW) {
    delay(50);
    if (digitalRead(coinSlotPin) == LOW) {
      coinsInserted++;
      tone(buzzerPin, 1000, 100);
    }
  }
  // Assign coins to a port
  for (int i=0; i<4; i++) {
    if (digitalRead(buttonPins[i])==LOW && coinsInserted>0)
      startCharging(i);
  }
  // Auto cutoff
  for (int i=0; i<4; i++) {
    if (relayEndTimes[i]>0 && millis()>=relayEndTimes[i])
      stopCharging(i);
  }
  updateLCD();
}
